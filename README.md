# rest_api_client

REST API client

Run using Python 3 with the command (default arguments are used if they are omitted):

python api_client.py API_KEY brand

DEPENDENCIES:
  scipy, statsmodels
  Intall using "python -m pip install scipy" and "python -m pip install statsmodels"
